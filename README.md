# ControleEstacionamento

ControleEstacionamento é um aplicativo simples para gerenciar a entrada e saída de veículos em um estacionamento.

## Versão 1.0.0
- Este projeto utiliza alguns conceitos de POO e MVC.

## Recursos
- Cadastro de preços
- Registro de entrada
- Registro de saída

## Tecnologias Utilizadas
- SQLite
- Entity Framework

## Instalação
- O aplicativo pode ser executado tanto utilizando o atalho chamado "ControleEstacionamento" na pasta raiz quanto iniciando o executável chamado "ControleEstacionamento.exe" localizado em `bin/release`.
